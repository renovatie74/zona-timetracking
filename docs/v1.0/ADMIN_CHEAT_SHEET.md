# Zona Time Tracker — Admin Cheat Sheet

**Version 1.0 · https://time.zonaproperties.ae**

| I want to… | Go to… |
|---|---|
| **Add a client** | Clients → Add Client |
| **Edit a client** | Clients → click client name |
| **Add a project** | Projects → Add Project |
| **Edit a project** | Projects → click project name |
| **Set project status** | Projects → click project → edit Status field |
| **Assign employees to a project** | Projects → click project → Assignments tab |
| **View project hours by week** | Projects → click project → Timesheet tab |
| **Mark a project week as invoiced** | Projects → project → Timesheet → mark week |
| **Add an employee** | Employees → Add Employee |
| **Edit an employee** | Employees → click employee name |
| **Activate / deactivate a user** | Employees → click employee → toggle Active status |
| **Resend activation email** | Employees → click employee → Resend Activation |
| **View an employee's timesheet** | Employees → click employee → Timesheet tab |
| **View an employee's Extras** | Employees → click employee → Extras tab |
| **View an employee's Mileage** | Employees → click employee → Mileage tab |
| **Add a team** | Teams → Add Team |
| **Check attendance** | Attendance (top nav) |
| **Find missing timesheets** | Dashboard → Missing Timesheets |
| **Review all Extras** | Extras (top nav) → filter by status |
| **Mark an Extra as processed** | Extras → click the entry → mark status |
| **Review all Mileage** | Mileage (top nav) → filter by week / employee / project |
| **Mark Mileage as complete** | Mileage → Mark Complete button |
| **Reopen a completed Mileage entry** | Mileage → Reopen button |
| **Export timesheet data (XLSX)** | Admin Console → Export → set date range → Download XLSX |
| **Change my password** | Profile (top-right avatar) → Change Password |
| **Sign out** | Profile (top-right avatar) → Sign Out |
