# Zona Time Tracker — Administrator Quick Start

**Version 1.0 · Production URL: https://time.zonaproperties.ae**

---

## Before Employees Start

### Clients
- [ ] Go to **Clients** → **Add Client**
- [ ] Enter client name, contact person, phone, email

### Projects
- [ ] Go to **Projects** → **Add Project**
- [ ] Set name, client, start date, status (`Active`)
- [ ] Note: projects with no assignments are accessible to all employees
- [ ] To restrict a project: open it → **Assignments** tab → add specific employees

### Teams (optional)
- [ ] Go to **Teams** → **Add Team** if you manage by team
- [ ] Teams are used to scope manager visibility

### Employees
- [ ] Go to **Employees** → **Add Employee**
- [ ] Enter first name, last name, email, role (`Employee` or `Manager`)
- [ ] Employee receives an **activation email** with a link to set their password
- [ ] Verify the employee can log in at https://time.zonaproperties.ae
- [ ] Ask employees to add the app to their phone home screen (PWA)

### Verify Before Go-Live
- [ ] All clients created
- [ ] All active projects created with correct status
- [ ] All employees created and activated
- [ ] Project assignments set where access should be restricted

## Daily Routine

| Task | Where |
|---|---|
| Check who attended | **Attendance** |
| Review project hours entered | **Employees** → employee → **Timesheet** |
| Review open Extras | **Extras** |
| Review open Mileage | **Mileage** |

---

## Weekly Routine

| Task | Where |
|---|---|
| Review each employee's week | **Employees** → employee → **Timesheet** |
| Check missing timesheets | **Dashboard** → Missing Timesheets |
| Mark Extras as Recorded/Processed | **Extras** → filter by Open |
| Mark Mileage entries as Complete | **Mileage** → Mark Complete |
| Reopen incorrect Mileage entries | **Mileage** → Reopen |
| Review project invoicing status | **Projects** → project → **Timesheet** |

---

## Billing Cycle

| Task | Where |
|---|---|
| Verify project hours per week | **Projects** → project → **Timesheet** |
| Mark weeks as invoiced | **Projects** → project → **Timesheet** → mark week |
| Export timesheet data | **Admin Console** → Export → select date range → Download XLSX |
| Review Extras for billing | **Extras** → filter by status |
| Review Mileage for billing | **Mileage** → filter by employee / project |
