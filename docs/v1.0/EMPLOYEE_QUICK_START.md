# Zona Time Tracker — Employee Quick Start

**App: https://time.zonaproperties.ae**

---

## First Login

1. **Open the activation email** from noreply@zonaproperties.ae
2. Click the link and **create your password**
3. Go to https://time.zonaproperties.ae and **sign in**
4. **Add to your phone home screen:**
   - iPhone: tap the Share button → "Add to Home Screen"
   - Android: tap the browser menu → "Add to Home Screen" or "Install App"
5. Go to **Account** tab → **Change Password** and set a personal password

---

## Every Day

### Set Your Attendance
1. Open the app → **My Day** tab
2. Tap **Set Attendance** (or edit if already set)
3. Enter your check-in time and check-out time
4. Save

### Add Project Hours
1. On **My Day**, tap **+ Add Project Hours**
2. Select the project
3. Enter hours worked
4. Save

> You can also view and edit the full week on the **My Time** tab.

---

## When Needed

### Add an Extra (own cost item)
1. Go to **Extras** tab → **+ Add**
2. Select **Own Cost**
3. Choose the project
4. Describe the item
5. Save — your manager will review it

### Add Mileage
1. Go to **Mileage** tab → **+ Add Entry**
2. Enter the date, project, kilometres driven
3. Add a note if helpful
4. Save — your manager will mark it complete

> You can only edit or delete entries that are **Open**. Once marked complete by admin, entries are locked.

---

## End of Week

- [ ] Check **My Time** — all days have attendance set
- [ ] Check **My Time** — project hours recorded for each working day
- [ ] Check **Extras** — any open items are correct
- [ ] Check **Mileage** — all trips recorded with the right project
- [ ] Correct any **Open** entries before your manager processes them
